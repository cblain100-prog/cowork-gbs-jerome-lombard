# Cowork GBS — Jérôme Lombard

Plugin Cowork pour Global Buying Services.

## Installation

1. Télécharger le fichier `gbs-jerome-lombard.skill` depuis les [Releases](../../releases)
2. Dans Cowork → Personnaliser → Plugins personnels → cliquer sur **+**
3. Téléverser le fichier `.skill`
4. Le plugin est actif dans toutes les conversations du projet

## Contenu

- Identité & entreprise GBS
- Client idéal & positionnement
- Offres & services (3 piliers)
- Charte graphique (couleurs, règles visuelles)
- Communication (ton, style, mots interdits)
- Templates (emails, LinkedIn, comptes-rendus, livrables)
- Stack technique
